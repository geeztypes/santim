How to Install

Step 1

Upload contents of the zip file to your web server's public directory. For example: www.yourdomain.com/css/webfont/

Step 2

Add contents of styles.css to your site's style sheet.

Step 3

Make sure you adjust the paths in code from styles.css to reflect the relative path on your server.
For this example you need to prepend /css/webfont/ to all src url definitions. Like this:
@font-face { font-family: 'Inku-Regular'; src: url('/css/webfont/Inku-Regular.eot?#iefix') format('embedded-opentype'), url('/css/webfont/Inku-Regular.woff') format('woff'), url('/css/webfont/Inku-Regular.ttf') format('truetype'), url('/css/webfont/Inku-Regular.svg#Inku-Regular') format('svg'); font-weight: normal; font-style: normal; }
